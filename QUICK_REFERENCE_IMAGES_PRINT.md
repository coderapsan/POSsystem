# 📸 Quick Guide: Adding Images & Printing

## Adding Menu Item Images (2 Ways)

### Method 1: Upload File 📁
1. Admin → Menu Manager
2. Click "Choose File" button
3. Select photo (JPG, PNG, WebP, GIF)
4. Preview appears ✓
5. Click "Add to menu"

**Tips**: 
- Max 5MB
- Use clear food photos
- 800x600px or larger works best

### Method 2: Image URL 🔗
1. Admin → Menu Manager
2. Paste image URL in "Image URL" field
3. Example: `https://example.com/momos.jpg`
4. Preview appears ✓
5. Click "Add to menu"

**Tips**:
- Must be publicly accessible
- Use full URL with https://
- Or use local path: `/images/momos.jpg`

## Editing Existing Items
1. Find item in menu list
2. Click **"Edit"** button
3. Change/add image using either method above
4. Click **"Save"**

## Printing Receipts 🖨️

### Desktop Computer
**Chrome / Firefox / Edge**:
- Click "Print Receipt" → Print dialog opens
- Select printer → Click Print
- Done! ✅

**Safari (Mac)**:
- Click "Print Receipt"
- If nothing happens: Press **⌘ + P**
- Select printer → Click Print

### iPad
**Safari**:
- Tap "Print Receipt"
- If dialog doesn't appear: Press **⌘ + P** (with keyboard)
- Or use Share button → Select "Print"

### iPhone / Android Phone
**Safari or Chrome**:
1. Tap "Print Receipt" button
2. **If nothing happens**:
   - Tap **Share button** (⬆️ icon)
   - Scroll down and select **"Print"**
   - Choose printer or Save as PDF
3. Done! ✅

## Common Issues & Quick Fixes

### Image Won't Upload
- ✗ File too big? → Compress to under 5MB
- ✗ Wrong format? → Use JPG or PNG
- ✗ Not working? → Try URL method instead

### Image Not Showing
- ✗ Check URL is public (not private Google Drive link)
- ✗ Clear browser cache: Ctrl+Shift+Delete
- ✗ Try different image

### Print Not Working
- ✗ **On mobile**: Use Share → Print
- ✗ **Popup blocked**: Allow popups in browser settings
- ✗ **Still stuck**: Try different browser

## Where Images Appear

Images you upload will show in:
- ✅ Menu browsing (when staff selects category)
- ✅ Item details popup
- ✅ Shopping cart (small thumbnail)
- ✅ Customer view (if enabled)
