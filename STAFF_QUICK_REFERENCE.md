# POS Quick Reference - New Features

## 🎯 Quick Start Guide for Staff

### Feature 1: Auto-Fill Customer Details

**When taking an order:**
1. Enter customer's phone number in the "Phone" field
2. **Wait 1 second** - system searches automatically
3. If customer exists → Name, Address, and Postal Code **auto-fill** ✨
4. Verify details with customer and adjust if needed
5. Continue with order

**Visual Cues:**
- 🔄 Spinning icon = System is searching
- 🆕 Green badge = New customer (first time)
- 🔄 Blue badge = Returning customer

---

### Feature 2: Address Lookup by Postal Code

**For delivery orders:**
1. Ask customer for postal code
2. Type it in the "Postal Code" field
3. **Wait 1 second** - system looks up address
4. Address field fills automatically with the area name
5. Ask customer for complete address and verify
6. Continue with order

**Example:**
- Type: `SW20 8LR`
- Auto-fills: "Kingston Upon Thames, Greater London"
- Then add: house/flat number and street name

---

### Feature 3: Customer Order History

**What you'll see:**
- After entering a phone number, a panel appears below
- Shows customer's previous orders
- Displays their order count and total spending
- See what they ordered before

**How to use it:**
- Check if they want "the usual"
- Recommend similar items
- Verify address matches previous orders
- Better customer service!

---

### Feature 4: Order Numbers

**Always check:**
- Order number appears as **#12345** (5 digits)
- Same number on screen, receipt, and printed copy
- Use this number to track orders
- Customer can reference it for inquiries

## ⚡ Time-Saving Tips

1. **Always enter phone number first** - it might fill everything else
2. **Let the system work** - wait for the spinning icon to finish
3. **Verify with customer** - auto-fill might need updates
4. **Check order history** - suggest their favorites
5. **Use postal code** - quick way to verify delivery area

## 🎨 What the Colors Mean

| Badge/Color | Meaning |
|------------|---------|
| 🆕 Green | New customer - first order |
| 🔄 Blue | Returning customer |
| ✅ Green "Paid" | Payment received |
| ⚠️ Yellow "Pending" | Payment not confirmed |
| 🔄 Spinning circle | System is searching |

## ❓ Troubleshooting

### "Customer details didn't auto-fill"
- They might be a new customer
- Phone number might be different than last time
- Just enter details manually - works normally

### "Address didn't auto-fill from postal code"
- Postal code might be typed wrong
- Internet connection might be slow
- Enter address manually - no problem

### "History panel not showing"
- Give it a moment to load
- Customer might be new
- System works normally without it
