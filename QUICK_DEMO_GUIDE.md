## 🆘 Quick Troubleshooting

### Problem: Dropdown doesn't appear
**Solution:**
- Check if you typed at least 5 characters
- Wait 0.6 seconds
- Check internet connection
- Try test code: CR4 3QR

### Problem: Wrong addresses shown
**Solution:**
- Re-type postal code correctly
- Add space if missing: CR4 3QR not CR43QR
- Some postcodes might not have data

### Problem: Keyboard navigation not working
**Solution:**
- Make sure dropdown is visible
- Click on postal code field first
- Try mouse click instead
